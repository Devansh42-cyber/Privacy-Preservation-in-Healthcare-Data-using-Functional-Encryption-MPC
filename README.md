# INSTALLATION INSTRUCTIONS:
# ============================================================================
# 1. Activate your virtual environment:
#    source venv/bin/activate
#
# 2. Install all dependencies:
#    pip install -r requirements.txt
#
# 3. For production (without dev tools):
#    pip install -r requirements.txt --no-dev
#
# 4. To update all packages:
#    pip install --upgrade -r requirements.txt
#
# ============================================================================
# NOTES:
# ============================================================================
# - TenSEAL requires CMake and build tools. Install with:
#   sudo apt install cmake build-essential libboost-dev
#
# - gmpy2 requires GMP library. Install with:
#   sudo apt install libgmp-dev libmpfr-dev libmpc-dev
#
# - All versions are tested and compatible with Python 3.10+
#
# - For minimal installation (core only), install just:
#   mpyc, tenseal, pycryptodome, numpy, pandas, pymongo
